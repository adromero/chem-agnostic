# CodelyTV per-rule PR scaffolds

Drafts for the three small PRs to open against
`CodelyTV/eslint-plugin-hexagonal-architecture` **if and only if** the
maintainers respond positively to the Discussion at
`docs/launch/codelytv-discussion.md`.

Each scaffold describes:
- The PR title.
- The PR body (paste into the GitHub PR description).
- The files the PR would add/modify in their repo.
- The corresponding source files in this repo to copy from.

**Do not open these PRs until the maintainers agree** — see
`docs/master-plan/track-s-launch-checklist.md` § CodelyTV contribution.

## Files

- `01-needs-interface.md`
- `02-no-concrete-class-import.md`
- `03-no-adapter-instantiation.md`

## Order

Open them in numbered order. If the first one is rejected for stylistic
reasons, fix the style before opening the others.
