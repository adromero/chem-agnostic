# Getting started with chemag

`chemag` (chemistry-agnostic) helps you describe and enforce the architecture
of your codebase using a small chemistry-inspired vocabulary: **compounds**
group related code, **units** play roles inside a compound, and **bonds**
declare which compounds are allowed to talk to which.

This walkthrough takes about two minutes. It covers:

1. **What is chemag?** — the one-screen mental model.
2. **Open a workspace** — point VS Code at a folder that has a
   `workspace.yaml` so the extension can activate.
3. **Run your first check** — invoke the `chemag: Check workspace` command
   and read the diagnostics in the Problems panel.

Use the steps in the **Get Started** sidebar on the right to walk through
each one in order. You can re-open this walkthrough at any time from the
command palette: **Help: Welcome → chemag: Get started**.
